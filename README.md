# Caster API

#### Backend-сервис на Node.js / Express / PostgreSQL. Основной упор сделан на надежность кода (автотесты) и автоматизацию окружения

### Стек:

* **Язык**: TypeScript (Node.js)
* **База данных**: PostgreSQL + Prisma ORM
* **Развертывание**: Docker Compose
* **Документация**: Swagger

### Особенности:

* **База данных**: Все запросы к базе реализованы через **Prisma**. Настроена автоматическая инициализация данных (seed)
* **Авторизация**: JWT. Для инвалидации токенов используется **Redis**. Сессии хранятся в БД
* **Тестирование**: Написаны интергационные автотесты (**Vitest/Supertest**) для проверки всех эндпоинтов
* **Docker**: Все зависимости (app, db, redis) изолированы и связаны через **Docker Compose**. Настроены проверки готовности (healthchecks)
* **Валидация окружения**: Реализована проверка обязательных переменных `env` при запуске. При отсутствии переменных выдаст ошибку

### Развертывание и запуск:

1. Поднять окружение:

`docker-compose up -d --build`

2. Подготовить базу для автотестов (для работы с env.test)

`npm run test:init`

3. Запустить тесты:

`npm run test`

### Доступные ресурсы:

* **Приложение**: `http://localhost:3000`
* **Swagger**: `http://localhost:3000/api-docs`
* **Adminer**: `http://localhost:8080`
